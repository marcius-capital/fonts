# Helvetica Neue Cyrillic

Free font with MIT licence

## How to start

1. Install in your project

```node
yarn add @marcius-studio/font
npm i @marcius-studio/font
```

2. Add in project

```js
// add in main.js
import '@marcius-studio/fonts'
```

```css
body {
    font-family: 'HelveticaNeueCyr', sans-serif;
	font-weight: 400;
}
```
